# About

Night Gym Academy is a premium educational platform focused on fitness science.

The goal is not to build a simple course website.

The goal is to create a complete educational ecosystem for students, coaches, athletes, and healthcare professionals.

---

# Vision

Build the world's most comprehensive fitness education platform.

---

# Mission

Deliver professional, scientific, practical, and easy-to-understand fitness education through modern technology and an exceptional learning experience.
