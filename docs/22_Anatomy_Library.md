# Purpose

This document defines the Anatomy Library of Night Gym Academy.

The goal is to build one of the world's most comprehensive anatomy education libraries for fitness professionals, students, coaches, athletes, and healthcare professionals.

Every anatomical topic must be scientifically accurate, professionally organized, and easy to understand.

---

# Goals

- Teach human anatomy scientifically.
- Explain muscle function.
- Improve exercise selection.
- Prevent injuries.
- Build strong scientific knowledge.
- Connect anatomy with practical training.
