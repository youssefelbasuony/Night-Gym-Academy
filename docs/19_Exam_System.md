# Purpose

This document defines the Examination System of Night Gym Academy.

The goal is to create a professional examination platform that evaluates knowledge, practical understanding, and coaching skills.

The examination system should simulate international certification standards while remaining fair, educational, and user-friendly.

---

# Goals

- Measure learning progress.
- Prepare students for professional certifications.
- Identify strengths and weaknesses.
- Encourage continuous learning.
- Reward achievement.
