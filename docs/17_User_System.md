# Purpose

This document defines the User System of Night Gym Academy.

The goal is to create a secure, scalable, and user-friendly account system that supports different user roles while protecting personal information and learning progress.

---

# User Types

The platform supports multiple user roles.

- Guest
- Student
- Parent
- Coach
- Instructor
- Admin
- Super Admin

Each role has different permissions.

---

# Guest

Can:

- Browse the homepage
- View public courses
- Read public articles
- Explore the platform

Cannot:

- Access lessons
- Take exams
- Earn certificates
- Save progress

---

# Student

Can:

- Register
- Login
- Enroll in courses
- Watch lessons
- Take quizzes
- Take exams
- Earn XP
- Earn certificates
- Save progress
- Bookmark lessons
- Download allowed resources
- Join community features (Future)

---

# Parent

Can:

- Approve child registration
- View child progress
- View certificates
- Receive notifications
- Monitor learning activity

Cannot modify educational records.

---

# Coach

Can:

- Access coach resources
- Create private notes
- Build training plans (Future)
- Access advanced educational tools

---

# Instructor

Can:

- Create courses
- Edit lessons
- Upload resources
- Create quizzes
- Create exams
- Review student performance

---

# Admin

Full management of:

- Users
- Courses
- Lessons
- Exercises
- Exams
- Certificates
- Community
- Reports
- Notifications

---

# Super Admin

Complete system control.

Includes:

- Platform settings
- Roles
- Permissions
- Database management
- Security settings
- System configuration

---

# Privacy

# Account Security

Features include:

- Email Verification
- Password Reset
- Remember Me
- Secure Sessions
- Device Management (Future)
- Two-Factor Authentication (Future)

---

# Child Protection

If the user's age is below the required minimum:

- Parent approval is required.
- Parent email verification is required.
- Parent account is linked.
- Child progress becomes visible to the parent.

---

# Account Status

Possible statuses:

- Active
- Pending Verification

- Suspended
- Banned
- Deleted
