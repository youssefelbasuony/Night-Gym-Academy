# Purpose

This document defines the Course System of Night Gym Academy.

The goal is to build a professional learning system that delivers structured, scientific, and engaging fitness education.

Courses should provide a clear learning path from beginner to professional.

# Goals

- Deliver structured education.
- Support long-term learning.
- Track student progress.
- Prepare students for professional certifications.
- Create an enjoyable learning experience.
