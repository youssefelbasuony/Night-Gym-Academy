# Development Rules

Always:

✅ Continue from the existing project.

✅ Follow the current design.

✅ Build reusable components.

✅ Keep the project scalable.

✅ Update documentation when necessary.

---

Never:

❌ Redesign completed pages.

❌ Change the color palette.

❌ Change typography.

❌ Rename classes without reason.

❌ Delete existing features without approval.

❌ Ignore the documentation.
