# Purpose

This document locks the visual identity of Night Gym Academy.

No AI model, developer, or designer is allowed to redesign the project unless the project owner explicitly approves the change.

The goal is to keep the entire platform visually consistent from the first page to the last page.

# Forbidden Changes

Do NOT:

❌ Change colors.

❌ Change typography.

❌ Change spacing.

❌ Change card style.

❌ Change button style.

❌ Change layout.

❌ Redesign completed pages.

✅ Add new features.

✅ Improve performance.
