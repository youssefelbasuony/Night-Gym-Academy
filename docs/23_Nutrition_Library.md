# Purpose

This document defines the Nutrition Library of Night Gym Academy.

The goal is to build one of the world's most complete nutrition education libraries.

The library should provide scientifically accurate, practical, and easy-to-understand information for beginners and professionals.

---

# Goals

- Teach nutrition scientifically.
- Help users build healthy eating habits.
- Support fitness goals.
- Provide practical meal planning.
- Improve long-term health knowledge.
